Trabalhando com ID e Classe

Seletores ID

É usado um seletor de id em um elemento HTML para deixar ele único e específico.

Deve ser usado uma única vez na pagina.

Exemplo

USANDO UM ID

Seletores de Clasee

O seletor de classe seleciona elemento com um atributo de classe específico.

Pode ser usado mais de uma vez na página

Exemplo:

USANDO CLASSE

USANDO CLASSE

USANDO CLASSE

OUTRO TEXTO
